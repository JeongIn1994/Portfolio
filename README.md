# 1.概要

## 1.1 使用目的
#### 本ウェブアプリケーションはPC,タブレット、モバイル環境から個人の開発履歴の記録、修正、削除、閲覧および
#### その開発中に発生した問題や得意事項などを記録、修正、削除、閲覧できる掲示板を重要機能を目的とする。
